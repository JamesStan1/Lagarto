# Charizze C. Lagarto — Portfolio (Next.js)

This is a simple personal portfolio built with Next.js showing Charizze C. Lagarto's projects, skills, education, and contact details.

## Features
- Home, About, Projects, Contact pages
- Responsive layout
- Simple components: Header, Footer, Hero, AboutSection, Skills, ProjectsList, ContactSection

Design updates:
- Improved typography (Poppins) and spacing for better readability
- Project cards include cover images and interactive buttons
- Responsive navigation with mobile menu
- Visual progress bars for percent-based skills

## Quick Start (Windows PowerShell)

1. Install dependencies
```powershell
cd c:\Users\Admin\Documents\GitHub\Lagarto
npm install
```

2. Run development server
```powershell
npm run dev
```

Open http://localhost:3000 in your browser.

## Build for production
```powershell
npm run build
npm start
```

## Deployment
You can deploy this Next.js app to Vercel, Netlify (using Next support), or any Node hosting that supports Next.
Short instructions for Vercel:
1. Install the Vercel CLI or sign into your Vercel account.
2. Run `npx vercel` and follow prompts.
